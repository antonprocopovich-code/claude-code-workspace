Пошаговая инструкция подключения:
1. Получить Access Token на figma.com
2. Добавить `FIGMA_ACCESS_TOKEN` в `~/.zshrc`
3. Добавить `mcpServers.figma` в `~/.claude/settings.json`
4. Получить File ID брендбука из URL Figma
5. Проверить через `/mcp`
