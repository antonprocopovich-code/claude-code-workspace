# Аудитория

> Стартовый файл. Обновить после прогона Стратега или Маркетолога.

## Основная персона: «Предприниматель-создатель»

| Параметр | Описание |
|---------|---------|
| Кто | 28–42 года, онлайн-предприниматель, эксперт, фрилансер |
| Уровень | Есть аудитория 1k–50k, но нет системы публикаций |
| Боль | Контент съедает время, хочет автоматизировать — но не знает как |
| Желание | Выходить регулярно без выгорания, не нанимая команду |
| Барьер | «AI — это сложно», «не технарь», «нет времени разбираться» |
| Триггер к действию | Видит что кто-то уже сделал — хочет так же |

## Вторичная персона: «Начинающий эксперт»

| Параметр | Описание |
|---------|---------|
| Кто | 22–32 года, строит личный бренд с нуля |
| Боль | Не знает о чём писать, нет голоса, боится быть неинтересным |
| Желание | Найти свой стиль и начать выходить системно |
| Барьер | Перфекционизм + непонимание что работает |

## Что аудитория НЕ хочет видеть

- Техно-жаргон без объяснений (API, prompt, token — без контекста)
- «Вот мой миллион» без доказательств процесса
- Советы без кейсов («надо делать Reels» — и всё)
- Длинные вводные без сути
