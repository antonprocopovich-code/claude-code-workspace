# Постоянные ограничения контента

> Применяются ко всем материалам без исключений. Обновлять только при реальном изменении позиции.

## Тематические ограничения

- **Не обещать гарантированный доход** от AI или контента
- **Не сравнивать себя с конкурентами** по имени — только по подходу
- **Не давать советов по инвестициям**, трейдингу, крипте
- **Не писать о личной жизни** если не связано с темой поста

## Форматные ограничения

| Платформа | Макс. длина | Обязателен CTA | Хэштеги |
|-----------|-------------|----------------|---------|
| Telegram | 800 слов | Да | Нет |
| Instagram | 2200 символов | Да | 3–5 |
| Shorts | 60 сек / ~150 слов | Да (словами) | 3 |

## Стилистические ограничения

- Не использовать слова-маркеры ИИ: «углубиться», «разобраться», «рассмотрим», «подводя итог», «в заключение»
- Каждый пост — одна идея, не несколько
- Хук в первых 2–3 строках — обязателен
- CTA конкретный («напиши в директ X», «сохрани пост»), не абстрактный («подпишись»)

## Юридические / репутационные

- Не публиковать чужой контент без разрешения
- Скриншоты переписок — только с согласия
- Упоминание клиентов — только с разрешения или анонимно
