# Постоянные ограничения контента — Антон Прокопович

> Источник: runtime/2026-06-16-brendbuk-v3.md (разделы 7, 10) + правила оркестратора
> Обновлён: 2026-06-22

## Тематические ограничения

- **Тема интимной близости — не замалчивать и не смягчать.** Это главная специализация и главный магнит аудитории. Говорить прямо.
- **Не сравнивать себя с конкурентами по имени** — только по подходу
- **Не давать советов по инвестициям**, трейдингу, крипте
- **Личная жизнь** — только если напрямую связана с темой поста и раскрывает путь Антона
- **Не обещать гарантированный результат** за конкретный срок

## Форматные ограничения по платформам

| Платформа | Тон | Длина | Эмодзи | Хэштеги |
|-----------|-----|-------|--------|---------|
| **Telegram** | Прямой, весомый — сразу суть | 300–800 знаков (можно больше если глубина требует) | Нет | Нет |
| **Instagram** | Чуть мягче, больше личного. Визуал несёт половину смысла. | 150–400 знаков в подписи | 1–2 максимум | 3–5 тематических |
| **TikTok** | Самый разговорный. Первые 2 сек решают всё. | Видео 45–90 сек | Нет в тексте | 2–3 широких |
| **Threads** | Как Telegram, но короче — одна острая мысль | 100–280 знаков | Нет | Нет |
| **ВКонтакте** | Теплее, больше сообщества — «свои» | 400–1000 знаков | Редко | 3–5 |

## Стилистические ограничения

- Не использовать слова-маркеры ИИ: «углубиться», «разобраться», «рассмотрим», «подводя итог», «в заключение», «давайте рассмотрим»
- Каждый пост — **одна идея**, не несколько
- **Хук в первых 2–3 строках** — обязателен. Лучший хук = то, что сложнее всего сказать вслух.
- **CTA конкретный** («напиши в директ X», «сохрани пост»), не абстрактный («подпишись»)
- Без восклицательных знаков в конце каждой мысли. Точка весомее.
- Три восклицательных знака подряд — запрещено
- Пассивный залог — избегать там, где возможен активный
- «Данный» / «является» / «осуществляет» — не использовать

## Запрещённые слова бренда

контент, продукт, аудитория, лиды, охваты, экспертность (как штамп), нетворкинг, прокачать, качать, работа над собой, помочь (в стиле «я помогу вам»), успешный успех, выйти из зоны комфорта, токсичный, ресурс (в значении «быть в ресурсе»), трансформируйся, прокачай отношения, инструменты счастья

## Визуальные ограничения (для Дизайнера)

- Не использовать `border-radius: 16px+` — разрушает архетип Правителя
- Не использовать `transition > 300ms` для интерактивных элементов
- Золото `#CA8A04` НЕ применять как основной текст на `#44403C` — недостаточный контраст
- Запрещены: орнаменты, иллюстративные emoji в дизайне — только SVG
- Запрещены: яркие фильтры Instagram, VSCO-пресеты, пастель, розовые/голубые тона, HDR
- Запрещены: восклицательный тон в дизайне, яркий красный CTA — противоречит Магу
- Субтитры в видео: Cormorant Garamond / Libre Baskerville, белый `#F2EDE6` на полупрозрачном тёмном оверлее
- Музыка в видео: инструментальная, низкий темп, без слов. Запрещено: клубная музыка, мотивационные треки с барабанами.

## Правила для Контент-Директора

1. **Каждый черновик** должен содержать хотя бы один элемент позиции Антона (7 столпов из brand-profile.md) — явно или косвенно
2. **Язык уровня 2–3** (из audience.md) — писать на языке того, что клиент знает но не говорит, или вообще не осознаёт. Не на языке открытых сомнений.
3. **Духовность + тело + отношения** — три столпа, которые всегда переплетены. Разделять их — упрощать.
4. **Шаблон хука** — взять самое сложное для произнесения → сделать точкой входа.

## Юридические / репутационные

- Не публиковать чужой контент без разрешения
- Скриншоты переписок — только с согласия
- Упоминание клиентов — только с разрешения или анонимно
