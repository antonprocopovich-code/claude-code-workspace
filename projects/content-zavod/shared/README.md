# shared/

Общий слой данных для субагентов.

**Оркестратор пишет → субагенты читают напрямую.**

## Что сюда кладёт оркестратор

| Файл | Когда создаётся |
|------|----------------|
| `brand-profile.md` | После прогона Стратега / Брендбук-агента |
| `audience.md` | После прогона Стратега или Маркетолога |
| `tone-of-voice.md` | После прогона Брендбук-агента |
| `constraints.md` | Вручную или после согласования с пользователем |

## Правила

- Файлы без дат — это эталоны, не история пайплайна
- Обновлять только при реальном изменении бренда или стратегии
- Субагенты читают отсюда сами — оркестратору не нужно прокидывать пути в каждом промпте
